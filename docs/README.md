## Core Concepts

### The Kernel
The orchestration engine that coordinates all agents using a Plan → Act → Review cycle.

### The 9 Agents
1. **Architect** - Service design
2. **Tester** - Test generation
3. **Infrastructure** - Docker/K8s/Terraform
4. **Docs** - Documentation
5. **Migration** - Database changes
6. **Analytics** - Monitoring setup
7. **DevOps** - Deployments
8. **Security** - Governance
9. **Kernel** - Orchestration

### Generated Services
Each service includes:
- Complete code (domain, API, database)
- 40+ test files (85%+ coverage)
- Docker/Kubernetes/Terraform
- GitHub Actions CI/CD
- Monitoring & observability
- API documentation
